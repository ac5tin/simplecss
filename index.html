Mobile only columns

To specify mobile sizes, use .col-*-m

.col-8 .col-5-m

.col-2 .col-5-m

          
    <div class="row">
      <div class="col-8 col-5-m">
          <div class="p-1 bg-grey1 b-black border round less-round border-thin text-center">
              <p>.col-8 .col-5-m</p>
          </div>
      </div>
  
      <div class="col-2 col-5-m">
          <div class="p-1 bg-grey1 b-black border round less-round border-thin text-center">
              <p>.col-2 .col-5-m</p>
          </div>
      </div>
    </div>

Fixed column width on mobile / desktop size

To force width to be the same on all sizes, use .col-*-stay

.col-4-stay

.col-6-stay

          
    <div class="row">
      <div class="col-4-stay">

Themes

Attach the theme name to the body element to apply theme.

Current Available Additional Themes

  • dark
          
    <body class="dark"></body>

mHide and mShow

.mHide hides content on mobile

.Show shows content on mobile

This line is hidden on mobile

This line is only visible on mobile

          
    <p class="mHide p-1 bg-grey1 border b-black">This line is hidden on mobile</p>
    <p class="mShow p-1 bg-grey2 t-white border b-black">This line is only visible on mobile</p>

Centering

3 types of centering

  • .center : centers block elements in most cases
  • .text-center : text align center
  • .flex-center : uses flex to center elements

Border Radius

For make an element rounded, just add the class 'round'

In additional, add

  • .less-round : to reduce the roundness
  • .very-round : to increase the roundness
